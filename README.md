# LabOptimizer

LabOptimizer is a machine learning optimization platform designed for research and engineering applications.

It provides:

- prediction model integration
- multiple optimization algorithms (random search, Bayesian optimization)
- experiment management
- visualization of optimization results
- production-ready architecture with FastAPI, Poetry, Docker, and CI/CD

## Tech stack

- FastAPI
- Poetry
- Optuna (coming next)
- Docker
- GitHub Actions
- Pytest
- Ruff
